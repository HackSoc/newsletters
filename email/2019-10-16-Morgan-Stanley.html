Coding Challenge
 
  
 
  

Morgan Stanley Coding Challenge is a half-day programming competition at the Morgan Stanley offices in Canary Wharf. By working with industry experts and under the guidance of a Morgan Stanley mentor, you’ll be able to build on your skills, expand your knowledge and gain an insight into the technology division. All you need are basic coding skills, enthusiasm, and willingness to learn.

The event will take place on Wednesday 23rd October, and Morgan Stanley will be covering some or all travel expenses for attendees. Applications close on 9am this Thursday so sign up now!

Break a Hacker's Mind - Web Application Vulnerabilities Exposed
 
  
 
  

6:30PM – 7:30PM, followed by pizza and networking

RCH/248X, Lakehouse, Floor 2, Ron Cooke Hub, Heslington East Campus

Join us for a special interactive guest lecture on Cyber Security with the Security Architecture team at Morgan Stanley.

Together we will work through a web application which is riddled with the common issues through which retail banking applications are compromised.

This interactive session will demonstrate how an attacker could exploit software vulnerabilities and explain remediation strategies.

This talk has been delivered at industry forums, universities and even the House of Lords. It is suitable both for those with and without technical experience in this space.

This session will include pizza and the opportunity for one-to-one Q&A with Morgan Stanley employees, including recent graduates.

This session is run in collaboration with HackSoc but is open to students from all science-degree backgrounds. For questions please contact hack@yusu.org or yorktech@morganstanley.com.
